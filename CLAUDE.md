# Claude Code Guidelines

## Changelog
After making changes, update `docs/CHANGELOG.md` under the `[Unreleased]` section. One-liner descriptions, categorized as Added/Changed/Fixed/Removed. When preparing a release, move `[Unreleased]` entries into a new `## [X.Y.Z] - YYYY-MM-DD` section.
