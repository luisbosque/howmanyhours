# Database Schema Export

This directory contains Room database schema export files, which are essential for:

## Purpose
- **Migration Tracking**: Enables safe database schema changes without data loss
- **Version Control**: Each schema version (1.json, 2.json, etc.) represents a database state
- **Migration Development**: Allows creating migration scripts between versions

## Files
- `com.howmanyhours.data.database.AppDatabase/1.json` - Database schema version 1
- Future versions will appear here when schema changes occur
